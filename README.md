# Orbital Debris Collision Detection System

## Overview

This project is a computational aerospace research prototype designed to analyze real satellite orbital data and identify potential close-approach events in low Earth orbit.
I used publicly available TLE datasets from CelesTrak, the system calculates satellite positions in real time, computes inter-object distances, and visualizes possible conjunction risks.

## Features
- Real-time satellite tracking
- Orbital position computation
- Collision proximity detection
- 2D orbital visualization
- Risk alert generation

## Technologies Used

- Python
- Skyfield
- NumPy
- Matplotlib

## Research Motivation

As the number of active satellites and orbital debris objects continues to increase, space situational awareness has become increasingly important for preventing collisions and maintaining sustainable orbital infrastructure.This project explores how publicly available orbital datasets can be used to model conjunction events and improve computational monitoring systems for orbital safety.
## Future Improvements

- Machine learning-based risk prediction
- 3D orbital simulation
- Real-time dashboard interface
- Predictive trajectory forecasting
- Autonomous collision avoidance algorithms
